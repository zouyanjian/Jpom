# Jpom 服务端接口文档

## 参与开发
```
# 安装 apiDoc
npm install apidoc -g
# 生成 apiDoc 文档（需要在项目根目录执行）
apidoc -c docs/apidoc.json -i modules/server/src/ -o docs/apidoc/doc-2.8.13
```
